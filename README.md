# Sacrifice Your Time

A dark, atmospheric 2D platformer where every step could be your last. Navigate through treacherous caves, dodge deadly traps, and make it out alive — if you can.

Built with **Godot 4** as a fork of the silksong-demo project.

---

## Controls

| Action | Key |
|---|---|
| Move Left / Right | `A` / `D` or `←` / `→` |
| Jump | `W` or `↑` |
| Double Jump | Press `W` / `↑` twice |
| Dash | `X` |
| Start Game | `Enter` |

---

## Features

- **Radial spotlight** — a small circle of light follows the player through the darkness. Stay alert!
- **Dynamic camera zoom** — the camera zooms out when you reach the floating platforms section so you can see what's coming
- **Wind zones** — certain areas push the player around, making jumps trickier
- **Crumbling platforms** — some platforms break under your feet, so keep moving!

---

## How to Run

1. Clone the repo:
   ```
   git clone https://github.com/mrsiuuuu-x1/sacrifice-your-time.git
   ```
2. Open **Godot 4**
3. Click **Import** and select the `project.godot` file
4. Hit **Play** and good luck!
